
# Hangman Game - League of Legends

Esta é uma aplicação feita em React que apresenta um jogo da forca com a temática de League of Legends para que você possa adivinhar o nome do campeão. Conectado a uma API simples com banco de dados MongoDB que salva o ranking de pontuação de cada jogador.
